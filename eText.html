QG TEXT ENGINE 1.0

Download html file


CC BY

This engine can be used to create multiple
choice text adventures games, with some RNG
elements and images too. It doesn't have a
UI so you'll have to edit the code directly,
but it's well documented and easy to use. If
you would like an example of what a game made
in this will look like, or how the files are
set up, you can check out our game "Adventures
of The Protagonist".
