Adventures of The Protagonist is a
simple text adventure game with some
RNG elements as well. The game starts
you in a mysterious house where not
all the paths make sense, or explain
themselves. Also known as
"The Existential Crisis Simulator"
and "A.O.T.P.", this game is mildy
entertaining. The home buttons will
also link to the old site, so sorry
about that.

Engine: QG TEXT ENGINE 1.0
Date Published: 2017 or 2018, I'm not sure
